서비스 소개

1. 서비스명

우리 동네 싱크홀맵

2. 서비스 목적

본 플랫폼은 시민이 도시 내 싱크홀 사고 이력을 실시간으로 확인할 수 있도록 하여 생활 속 안전 인식 제고와 사고 예방에 기여합니다.

또한 공공기관에는 정책 수립 및 유지보수 예산의 효율적 집행을 위한 의사결정 기반을 제공합니다.

아울러 부동산 수요자들이 매물 지역의 지반 안전성을 사전에 확인할 수 있도록 지원함으로써 신뢰도 높은 주거지 선택과 시장 정보 비대칭 해소에도 기여하고자 합니다.

3. 주요 기능

  • 싱크홀 위치 시각화 : 지도 기반으로 발생 지점을 마커로 표시
  • 사고 상세 정보 팝업 : 마커 클릭 시 사고일, 원인, 지층 등 정보 제공
  • 포상제도 연계 안내 : 안전신문고, 지자체 포상제도 링크 제공
  • 데이터 통계 시각화 : 지역별/시간대별 침하 발생 통계 제공
  • (예정) 예측 기능 설계 : 위치 기반 침하 위험도 예측

4. 데이터 수집

  • 데이터 : 지하안전정보시스템(JIS), 공공데이터포털, 지오빅데이터 오픈 플랫폼

※ 본 서비스는 공공데이터 기반으로 구축되었으며, 데이터는 지속적으로 업데이트됩니다.
